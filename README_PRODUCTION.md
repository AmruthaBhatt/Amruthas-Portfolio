## 🔧 Customization

### Update Projects

Edit `script.js` and modify the `projects` array:

```javascript
const projects = [
    {
        id: 1,
        title: "Your Project",
        description: "Project description here",
        link: "https://project-link.com"
    },
    // Add more projects...
];
```

### Update Personal Info

Edit `index.html`:
- Change name in hero section
- Update tagline
- Modify skills in About section

### Customize Colors

Edit `style.css`:
- Primary color: `#667eea`
- Secondary color: `#764ba2`

## 📤 Deploy Updates
